Question-1: What are the different ways to select an element in the DOM?

Answer: The different ways to select an element in the DOM are:

  • getElementById
  • getElementsByClassName
  • getElementsByTagName
  • querySelector
  • querySelectorAll
  • getElementsByName

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer: The difference between innerHTML, innerText, and textContent are:

innerHTML:

  • Returns or sets the HTML content inside an element.
  • Parses and renders HTML tags.

innerHTML:

  • Returns or sets only the visible text inside an element.
  • Ignores hidden elements

textContent:

  • Returns or sets all text, including hidden text.
  • Does not parse HTML tags, treats them as plain text.

Question-3: What is event delegation in the DOM?

Answer: Event delegation in the DOM is:

  • Uses event bubbling to allow a parent element to handle events for its child elements.
  • Improves performance and works for dynamically added elements.

Question-4: What is event bubbling in the DOM?

Answer: Event bubbling in the DOM is:

  • Event bubbling is a process where an event starts from the target element and propagates up to its parent elements.
  • It allows handling events at a higher level instead of adding listeners to multiple child elements.

Question-5: How do you create, add, and remove elements using JavaScript?

Answer:

  • Create an element using appendChild().
  • Add an element using document.createElement().
  • Remove an element using remove().
